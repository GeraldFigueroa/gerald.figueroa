## Descripción

Este proyecto consiste en una aplicación web diseñada para permitir registrar palabras personalizadas junto con su descripción y un color representativo. La idea principal es ofrecer un **diccionario personal** donde se puedan almacenar términos relevantes para su **aprendizaje**, trabajo o referencias personales.

## Funcionamiento
1. **Registro de Palabras**: Permite ingresar una nueva palabra junto con su descripción y seleccionar un color representativo.
2. **Almacenamiento**: Una vez registrada, la palabra se guarda en Firebase, asegurando disponibilidad y persistencia de los datos.
3. **Visualización de Tarjetas**: Todas las palabras registradas se muestran en tarjetas en la parte inferior de la página utilizando el color asignado.
4. **Interacción con Hover**: Al posicionar el cursor sobre una tarjeta, se despliega la descripción resaltando visualmente la información.

## Tecnologías
Tecnologías Utilizadas
- **ReactJS**: Framework utilizado para el desarrollo de la interfaz de usuario, garantizando una experiencia fluida e interactiva.
- **Firebase**: Plataforma de backend utilizada para el almacenamiento de datos en tiempo real, permitiendo una gestión eficiente de las palabras registradas.
- **HTML y CSS**: Lenguajes utilizados para la estructura y estilización de la interfaz de usuario, asegurando un diseño atractivo y adaptable.
